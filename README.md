# 練習自造組件

作者：我


## 安装

使用本框架前，請在 CSS 中開啟 border-box

```
*{ box-sizing: border-box; }
```
所有組件變數
```
html {
    --font-size: 14px;
    --color: rgb(5, 5, 5);
    --button-height: 32px;
    --button-bg: white;
    --button-active-bg: #eee;
    --border-radius: 1px;
    --border-color: #999;
    --border-color-hover: #666;
    }
```
